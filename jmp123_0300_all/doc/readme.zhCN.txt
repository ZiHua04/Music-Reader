作者更改或删除网络搜索模块时你不会收到通知。


3.如何编译、启动播放
------------------------------------------------------------------------
(1)编译
windows 平台用户运行 build.cmd 批命令脚本；
linux 平台用户运行 build.sh 批命令脚本；

(2)启动GUI播放器
java -jar jmp123.jar
或
java -cp jmp123.jar jmp123.gui.Player

启动过程自动加载user.dir下的default.m3u播放列表文件。

(3)启动控制台迷你播放器
java -cp jmp123.jar jmp123.Play <文件名>
指定的文件名可以是本地磁盘文件，也可以是一个网络文件。


5.解码速度测试(只解码不播放)
------------------------------------------------------------------------
java -cp jmp123.jar jmp123.test.Decode <文件名>

指定 JVM 参数，解码速度更快：
java -server -XX:+AggressiveOpts -XX:+UseSpinning -cp jmp123.jar jmp123.test.Decode <文件名>

注：(1)由于解码速度很快，指定的MP3文件播放时长30分钟以上测试结果更可信。
(2)指定的文件名只能为本地磁盘MP3文件。
